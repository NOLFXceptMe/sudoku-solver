## sudoku-solver

Solves sudoku puzzles in a simplified SDK file format. 
Usage
--
`ruby sdksolver.rb puzzle.sdk`

Example
--

```
└─[$] <git:(master*)> cat problem1.sdk  
1..489..6
73.....4.
.....1295
..712.6..
5..7.3..8
..6.957..
9146.....
.2.....37
8..512..4

└─[$] <git:(master*)> ruby sdksolver.rb problem1.sdk

7|3|.|.|.|.|.|4|.
.|.|.|.|.|1|2|9|5
```
